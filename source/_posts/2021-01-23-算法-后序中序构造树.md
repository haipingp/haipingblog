## 二、问题思路和算法
* 1. 根据后序找到最后一个为根节点 2. 根据根节点在中序中的位置，找到左右子树对应的节点 3. 对左右子树分别递归进行1 2两步
* 代码
``` cpp
class Solution {
public:
    TreeNode* buildTree(vector<int>& inorder, vector<int>& postorder) 
    {
        return buildTree(inorder, 0, inorder.size()-1, postorder, 0, postorder.size()-1);
    }
    
    TreeNode* buildTree(vector<int>& inorder, int in_start, int in_end, vector<int>& postorder, int post_start, int post_end)
    {
        if (in_start > in_end || post_start > post_end)
            return nullptr;

        TreeNode* root = new TreeNode(postorder[post_end]);
        auto it = find(inorder.begin()+in_start, inorder.begin()+in_end, root->val);
        int rootpos = it - inorder.begin();
        int numleft = it - inorder.begin() - in_start; 

        root->left = buildTree(inorder, in_start, rootpos-1, postorder, post_start, post_start+numleft-1);
        root->right = buildTree(inorder, rootpos+1, in_end, postorder, post_start+numleft, post_end-1);
        return root;
    }
};
```
