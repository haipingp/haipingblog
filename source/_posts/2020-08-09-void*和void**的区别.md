## 区别
* `void*`指向无类型的内存，`void**`指向`void*`
* 任意类型的指针都可以转换为`void *`，例如，`void**`就可以转换为`void*`
* `void**`常用于指向`void*`类型的数组，而C++里可以用`vector<void*>`表示，因此`void**`也不常用

## 类型转换关系的区别
* `void*`可以与任意类型指针相互转换，而`void**`不可以
* 可以转换的关系：
``` cpp
char* -> void*
void* -> void**
void** -> void*
void* -> char*
```
* 不可以转换的关系
``` cpp
char* -> void**
void** -> char*
```
